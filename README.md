# Next.js, TypeScript with next-sass and CSS-Modules

(https://github.com/zeit/next-plugins/tree/master/packages/next-sass)

Use bootstrap reboot for normalize css

#### Start development

1.  npm i
2.  npm run dev
3.  open (localhost:3001)

#### Start production build

1.  npm i
2.  npm run build
3.  npm run start
